# Bus Ticket Python Project

## Project Overview
This project is a Python based Bus Ticket Purchase System.
The system allows users to view and purchase bus tickets using data
stored in a CSV file.

The project is developed as part of the Foundations of Computer
Programming module coursework.

---

## Project Objective
The main objective of this project is to demonstrate:
- Basic Python programming skills
- Reading data from CSV files
- Taking user input
- Using conditional statements and loops
- Handling invalid user input
- Simple file handling
- Using GitHub for version control

---

## Users of the System
The system has two types of users:

### 1. Customer
- View available bus tickets
- View ticket prices
- Purchase a ticket (basic simulation)

### 2. Administrator
- Update ticket prices
- Manage ticket information

---

## Data Source
The data used in this project is stored in a CSV file.
The file contains information about:
- Ticket types
- Ticket names
- Prices
- Ticket conditions

Unnecessary columns have been removed to simplify the data.
